## keyShape

- **Type**:

```typescript
StyleProps & {
  /** Radius of the circle */
  r: number;
};
```

The related circle style can be referred to in [`CircleStyleProps`](../shape/CircleStyleProps.en.md).

- **Default**:

```json
{
  "r": 16
}
```

- **Required**: No
