## keyShape

- **Type**:

```typescript
StyleProps &
  {
    /**
     * width
     */
    width: number,
    /**
     * height
     */
    height: number,
  };
```

The related rect style can be referred to in [`RectStyleProps`](../shape/RectStyleProps.en.md).

- **Default**:

```json
{
  "width": "32",
  "height": "32"
}
```

- **Required**: No
