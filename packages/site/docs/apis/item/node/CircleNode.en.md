## keyShape

- **Type**:

```typescript
StyleProps & {
  /** radius of circle */
  r: number;
};
```

For more detailed style configuration, refer to [Circle](../shape/CircleStyleProps.en.md)。

- **Default**:

```json
{
  "r": 16
}
```

- **Required**: No
