## KeyShapeStyle

相关的图形样式参考 [`CircleStyleProps`](../../shape/CircleStyleProps.zh.md), 扩展属性如下：

### KeyShapeStyle.r

圆的半径。

- **是否必须**：否
- **类型**：`number`
- **默认值**：`16`
