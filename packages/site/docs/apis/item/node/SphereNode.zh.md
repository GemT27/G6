## KeyShapeStyle

### KeyShapeStyle.r

球半径。决定了球体的大小。

- **是否必须**：否
- **类型**：`number`
- **默认值**：`5`

### KeyShapeStyle.latitudeBands

球体的纬度带数量。这个数值影响球体的详细程度和渲染质量。

- **是否必须**：否
- **类型**：`number`
- **默认值**：`32`

### KeyShapeStyle.longitudeBands

球体的经度带数量。同样影响球体的详细程度和渲染质量。

- **是否必须**：否
- **类型**：`number`
- **默认值**：`32`
